# 抽奖小Demo

## 食用须知

运行本项目你需要：

```sh
git clone https://github.com/llhongwei/LuckyDraw.git
```

然后安装相应依赖

```sh
npm install
```

然后可以使用以下命令实现热重载运行：

```sh
npm run serve
```

使用以下命令可以实现生产环境文件：

```sh
npm run build
```
